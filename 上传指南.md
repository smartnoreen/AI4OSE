# 上传到 GitHub 指南

## 方法一：使用 GitHub Web 界面（推荐）

1. **创建新仓库**
   - 访问：https://github.com/new
   - Repository name: `AI4OSE`
   - Description: `rCore ch5 测试实现`
   - Public/Private: 根据需要选择
   - 点击 "Create repository"

2. **上传文件**
   - 在新创建的仓库页面，点击 "uploading an existing file"
   - 将 `ch5测试` 文件夹中的所有文件拖拽到网页
   - 填写 commit message: "Initial commit: rCore ch5 implementation"
   - 点击 "Commit changes"

## 方法二：使用命令行

### 前提条件
需要配置 GitHub 认证（Personal Access Token）

### 步骤

1. **在 GitHub 创建仓库**
   - 访问：https://github.com/new
   - Repository name: `AI4OSE`
   - 点击 "Create repository"

2. **获取 GitHub Token**
   - 访问：https://github.com/settings/tokens
   - 点击 "Generate new token (classic)"
   - 勾选 `repo` 权限
   - 生成并复制 token

3. **推送代码**
```bash
cd ch5测试

# 初始化 git（如果还没有）
git init
git add .
git commit -m "Initial commit: rCore ch5 implementation"

# 添加远程仓库（使用 token）
git remote add origin https://YOUR_TOKEN@github.com/smartnoreen/AI4OSE.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

## 方法三：使用 SSH

### 前提条件
需要配置 SSH 密钥

### 步骤

1. **配置 SSH 密钥**（如果还没有）
```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
cat ~/.ssh/id_ed25519.pub  # 复制公钥
```

2. **添加 SSH 密钥到 GitHub**
   - 访问：https://github.com/settings/keys
   - 点击 "New SSH key"
   - 粘贴公钥并保存

3. **推送代码**
```bash
cd ch5测试

# 初始化 git（如果还没有）
git init
git add .
git commit -m "Initial commit: rCore ch5 implementation"

# 添加远程仓库（使用 SSH）
git remote add origin git@github.com:smartnoreen/AI4OSE.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

## 常见问题

### Q: 推送时要求认证？
A: 使用 Personal Access Token 或 SSH 密钥

### Q: 出现 "repository not found" 错误？
A: 确保已在 GitHub 上创建了 AI4OSE 仓库

### Q: 出现 "permission denied" 错误？
A: 检查 token 或 SSH 密钥权限
